
# Visualizaciones_Interactivas

### Curso
Visualización de Información (IC8056)

---

### Propósito
Este proyecto tiene como objetivo crear visualizaciones interactivas utilizando el enfoque **Open Data Visualizations** que consiste en generar
imágenes SVG que incluyan toda la fuente de datos utilizada para generar las gráficas.

### Esquema de Folders

```
Visualizaciones_Interactivas/
│
├── Datos/              # Datos a utilizar en las gráficas
│   └── *.json
│
├── Plantillas/         # Código inja para generar los gráficos
│   └── *.inja
│
├── Imagenes/           # Imágenes svg con metadata
│   └── *.svg
│
├── Paginas/            # Páginas de cada gráfica
│    └── *.html         
│
├── index.html          # Página principal que conecta las páginas
├── simple-table.css    # Estilos para la tabla interactiva
├── simple-table.js     # Interactvidad con la tabla
└── svg-controls.js     # Interactvidad con los svg 


```

### Dirección de la Página Principal

Las visualizaciones interactivas se encuentran en la siguiente dirección: https://dayanaxie.github.io/Visualizaciones_Interactivas/
